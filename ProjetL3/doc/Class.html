Contenu des classes du jeu

    Game

    Contient tout les éléments présents dans le jeu, que ce soient les écrans d'accueil, d'options, le jeu, le profil du joueur...

  • -Etat du jeu (menu principal, options, quitter, chargement, en jeu, en pause...)
  • -Liste des ecrans du jeu
  • -Profil du joueur
  • -Horloge interne : Une simple sf::Clock, reinitialisée à chaque update.
