git init : intilialise un nouveau dépôt. Transforme un repertoire en dépôt Git acceptant les commandes Git.

git config : Configurer un dépôt.

git help : Aide

git status : Statut du dépôt.

git add nomFichier : Inclus un fichier dans les instantanés du Git.

git commit -a -m "Message" : Saisie un instantané du dépôt.

git branch nomBranche : Crée une nouvelle branche.

git checkout nomBranche : Rapatrie une branche d'un dépôt.

git merge nomBranch : Fusionne la branche.

git push nom branche : Envoi les modifications sur Github.

git pull nom branche : Récupère la dérnière version de Github.

mettre à jour :
git pull projet master
envoyer :
git add . : Ajouter tout les nouveau fichier et les fichiers mis a jours.
git commit -m Message : Preparer le pack à anvoyer sur github
git push projet master : Envoyer le pack sur github
